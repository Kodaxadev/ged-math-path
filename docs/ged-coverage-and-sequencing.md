# STEP GED Math Coverage and Sequencing

## Purpose

STEP is not a general worksheet library. It is a procedure-first learning path for a learner who can understand the question but can lose the thread when a math method introduces unexplained symbols, surprise numbers, too many visible steps, or too much arithmetic at once.

This document keeps the course honest: what is already taught, what is partly covered, what still needs to be built, and why the sequence is structured around test tools rather than textbook order.

## External checklist reference

The public Get Sum Math GED Math study-guide page was used as an external topic-inventory and test-tool sequencing reference. Its public page groups GED math study around:

- beginner/easier-entry topics;
- calculator-assisted problem types;
- formula-sheet problem types;
- no-calculator problem types;
- broader GED topic practice and full practice tests.

STEP does **not** copy Get Sum Math worksheet questions, images, videos, answer keys, or presentation flow. STEP creates its own problems and its own instructional method.

## STEP teaching rule

Every topic taught in STEP must follow these rules:

1. One lesson teaches one procedure family.
2. One practice problem uses the procedure just shown.
3. A new number is never introduced without showing where it came from.
4. A problem reveals one step at a time only after the learner attempts a move.
5. Walkthrough results may inform the mistake journal, but only independent practice counts toward mastery.
6. Calculator use begins with what to write before touching the calculator.
7. Formula-sheet use begins with finding and copying the formula before inserting numbers.

## Coverage states

The GED Map uses three states:

- **Ready in STEP** — a relevant lesson exists and can be started from the map.
- **Partly built** — some of the procedure family is covered, but authentic formats or related variants remain to build.
- **To build** — intentionally queued; it should not silently appear to be covered.

## Test-tool pathways

### Fast confidence wins

Purpose: reduce start friction with shorter, clearer problem families before harder multi-step procedures.

Current anchors:

- Mean, median, mode — `data-averages`
- Rates — `rates-speed`
- Functions / substitutions — `function-lines`
- Percent change — `percent-change`

### Calculator helps here

Purpose: teach the learner to identify and write the calculation, then use the TI-30XS for heavy arithmetic.

New route lesson:

- `calculator-entries` — write the expression first, then type it.

Current linked families include percents, probability, functions, line equations, quadratics, and calculator word problems. Simple interest, weighted average, and answer-choice calculator strategies remain queued.

### Formula sheet route

Purpose: use the provided GED formula sheet without requiring memorization as a prerequisite.

New route lesson:

- `formula-sheet-route` — find the shape, copy the formula, fill values, calculate, label units.

Current linked families include area, volume, Pythagorean theorem, and part of circumference/perimeter. Surface area remains queued.

### No-calculator foundation

Purpose: make the first non-calculator portion manageable through compact written moves instead of mental calculation overload.

New route lesson:

- `no-calculator-route` — short move, calculate, check, continue.

Current linked families include fractions, exponents/roots, equations, decimals and order of operations. Number lines, undefined expressions, rational exponents, and polynomials remain queued.

### Mixed GED skills

Purpose: move from isolated procedures to identifying which procedure a GED-style question needs.

Current anchor:

- `mixed-check`

Additional mixed-format question banks should be generated only after high-frequency procedure families have stable lesson feedback.
